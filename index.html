📌 Project Overview

Designed and developed a scalable Selenium automation framework to validate end-to-end e-commerce workflows including login, product search, cart, and checkout functionality.

🚀 Automated 50+ test cases and reduced manual regression effort by 60%.

🧪 Test Coverage

  • Cross-browser compatibility (Chrome, Edge, Firefox)
  • User authentication validation
  • Product search and filtering
  • Add-to-cart and checkout flow
  • Regression test suite execution

🏗 Framework Architecture

  • Page Object Model (POM) design pattern
  • Reusable utility classes and base test setup
  • TestNG for test execution and grouping
  • Maven for dependency management
  • HTML reporting integration
